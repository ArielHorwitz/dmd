  Recap the session in detail for the user who has been away for some
  time and needs to be brought back up to speed.
argument-hint: "What do you need to be brought up to speed on?"
disable-model-invocation: true
---

The user has been away for some time and  is only now returning to
the session.

Write a recap of the session in detail:
- Provide a short and concise "history" of the session (how did we get here)
- Provide an overview of all the major concepts, tradeoffs, and design
  decisions that we considered and arrived at
- Describe the current state of things including changes and decisions
- Suggest what would be the next steps that would/should be taken

If the user passed arguments, consider them as what to put more or less
emphasis in your recap, and what to focus on.
